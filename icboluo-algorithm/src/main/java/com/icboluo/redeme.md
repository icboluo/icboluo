## 树结构

查询、增加效率都比较高

树的一些 描述
```
节点、根节点、父节点、子节点、叶子节点、节点的权（val）
路径（根->该节点）、层、子树、树的高度、森林
```

二叉树：每个节点最多只有2个子节点

满二叉树：叶子节点都在最后一层、最后一层是满的（2.n-1）

完全二叉树：最后一层叶子节点左边连续，倒数第二层右边连续
